# Privacy Policy

Effective: July 19, 2026

Chat Exporter by Tom Raz processes AI conversation content locally in the user's browser.

## Data handled

After the user clicks the extension, it temporarily reads the conversation content loaded in the active tab. It may also read the page title, detected model and current URL. The URL is included in an export only when the user selects that option.

## Storage and transmission

- Conversation content is not stored by the extension.
- Conversation content, page URLs and browsing activity are not transmitted to Tom Raz or a third party.
- The extension makes no network requests and contains no analytics, ads or remote executable code.
- The only persisted setting is the English/Hebrew interface preference in local browser storage.
- Exported files are saved locally through the browser; clipboard access occurs only after the user presses Copy.

## Permissions

- `activeTab`: temporary access to the tab where the user invokes the extension.
- `scripting`: execution of the extractor bundled in the reviewed package.
- `storage`: local language preference.

Chat Exporter does not sell, share or use user data for advertising, profiling, credit decisions or unrelated purposes. Material changes will be recorded in the changelog and store privacy disclosures.
