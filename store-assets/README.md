# Store assets

Generated with `npm run assets:store` from the Remotion compositions in `media/`.

- `en/01.png` through `en/05.png`: English 1280×800 screenshots.
- `he/01.png` through `he/05.png`: Hebrew 1280×800 localized screenshots.
- `promo-small-440x280.png`: Chrome/Edge small promotional tile.
- `promo-marquee-1400x560.png`: optional large promotional tile.

The screenshots use an anonymous demonstration conversation and faithfully represent the extension's current interface and declared behavior. No personal conversation or account data is included.
