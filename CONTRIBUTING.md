# Contributing

Keep contributions focused on the extension's single purpose: exporting AI conversations locally.

1. Do not add analytics, advertising, remotely hosted code or broader permissions.
2. Add or update an anonymized fixture when changing a platform adapter.
3. Run `npm run check` before opening a pull request.
4. Never commit real conversation exports, credentials, cookies or access tokens.
5. Describe platform DOM changes and the browser versions used for manual validation.
