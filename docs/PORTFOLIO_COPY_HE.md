# Chat Exporter

Chat Exporter הוא תוסף דפדפן מקומי לייצוא שיחות מ־ChatGPT, Claude, Gemini, Copilot ו־Perplexity. הוא שומר את מבנה השיחה כ־Markdown או כטקסט פשוט, מאפשר העתקה ללוח ומציג אם שלמות השיחה אומתה. כל העיבוד מתבצע בדפדפן, ללא חשבון, אנליטיקה או העלאת תוכן לשרת.

**תגיות:** חינם · קוד פתוח · מקומי ופרטי · Chrome / Edge / Firefox

**כפתור ראשי לפני פרסום בחנויות:** צפייה בקוד

**קישור:** https://github.com/talktome8/chat-exporter

**כפתורי החנויות:** יוצגו רק לאחר שכל עמוד חנות יהיה פעיל.

### נתונים מהירים

| שדה | תוכן |
|---|---|
| סוג כלי | תוסף דפדפן, Manifest V3 |
| פלט | Markdown, טקסט והעתקה ללוח |
| פרטיות | עיבוד מקומי, ללא שרת וללא אנליטיקה |
| תמיכה | Chrome, Edge ו־Firefox |

> **פרטיות כברירת מחדל**  
> Chat Exporter קורא את השיחה רק לאחר לחיצה יזומה, מעבד אותה במכשיר ושומר את הקובץ ישירות למחשב. תוכן השיחה אינו נשלח למפתח.

## הבעיה

שיחות AI ארוכות מכילות לעיתים קוד, טבלאות, רשימות וקישורים שקשה לשמור באמצעות העתקה ידנית. התוצאה עלולה להיות חלקית, לא מסודרת וללא חלוקה ברורה בין המשתמש למודל.

## המגבלות

מבנה האתרים משתנה לאורך זמן, וחלקם טוענים הודעות ישנות רק בזמן גלילה. לכן התוסף אינו מבטיח שכל שיחה תיטען מיד או שכל אתר AI יתמוך בכל רגע. כאשר לא ניתן לאמת שלמות, המשתמש מקבל סימון ברור במקום הבטחה מטעה.

## הפתרון והבנייה

פיתחתי מנגנוני חילוץ נפרדים לפלטפורמות הנתמכות, עם ייצוא מהיר של ההודעות שכבר נטענו ואפשרות לבדוק ולטעון את השיחה המלאה. התוסף משמר Markdown, קוד, רשימות, קישורים וטבלאות, תומך בעברית וב־RTL ומבקש רק שלוש הרשאות מצומצמות: גישה זמנית לכרטיסייה הפעילה, הרצת מנגנון החילוץ ושמירת העדפת השפה.

## מאפיינים

- ייצוא מהיר של הודעות שכבר נטענו.
- בדיקת שיחה מלאה עם ביטול, מגבלת זמן ושחזור מיקום הגלילה.
- מצבי שלמות: מלא, חלקי או לא מאומת.
- Markdown, טקסט פשוט והעתקה ללוח.
- סינון הודעות משתמש ותשובות AI.
- ממשק אנגלית ועברית עם RTL.
- ללא חשבון, פרסומות, אנליטיקה, שרת או קוד מרוחק.
- תמיכה רשמית ב־ChatGPT, Claude, Gemini, Copilot ו־Perplexity; תמיכת Beta ב־Grok וב־Mistral.

## הערת פרסום

לפני אישור החנויות יש להציג "בקרוב" במקום כפתור התקנה. לאחר הפרסום יתווספו קישורים נפרדים ל־Chrome Web Store, ל־Microsoft Edge Add-ons ול־Firefox Add-ons.
